# 🎓 AI-Powered Personalized Learning Recommendation System for Engineering Students

## 📌 Project Overview

The AI-Powered Personalized Learning Recommendation System is an intelligent career guidance platform designed to help engineering students improve their placement readiness, identify skill gaps, and receive personalized learning recommendations.

The system leverages Machine Learning, Data Analytics, and Interactive Visualization to predict placement opportunities, estimate expected salary packages, evaluate student readiness, and generate customized career roadmaps.

This project aims to bridge the gap between academic performance and industry requirements by providing actionable insights for career development.

---

# 🚀 Problem Statement

Many engineering students face challenges such as:

* Lack of awareness about placement readiness
* Difficulty identifying skill gaps
* No personalized career guidance
* Limited understanding of industry expectations
* Uncertainty regarding salary potential

This project addresses these challenges using AI-driven recommendations and predictive analytics.

---

# 🎯 Objectives

* Predict placement probability
* Estimate expected salary package

# 🤖 Machine Learning Models

## Placement Prediction Model

Purpose:

Predict whether a student is likely to get placed.

Input:

Student profile features

Output:

* Placement Probability (%)
* Placement Status

Algorithms Used:

* Logistic Regression
* Random Forest Classifier
* Gradient Boosting

---

## Salary Prediction Model

Purpose:

Estimate expected salary package.

Output:

Expected Salary (LPA)

Algorithms Used:

* Linear Regression
* Random Forest Regressor
